A React Quiz with only useReducer used. in this web app there is a json server for the data of questions and users so make sure to (npm run server) you can add or remove a user, and the highscore of each user gets updated when ever their is a new highscore. all the new data gets saved or updated in the data.json file so that when you reload the page it gets the data from their when json server is running
